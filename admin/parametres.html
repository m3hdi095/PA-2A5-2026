Stripe

Les clés Stripe sont des secrets serveur : elles ne transitent jamais par le navigateur.
Configurez-les dans le fichier api-go/.env : STRIPE_SECRET_KEY=sk_live_...

Le secret JWT est une clé serveur confidentielle : l'afficher ici serait une faille de sécurité.
Modifiez-la dans api-go/.env : JWT_SECRET=votre_secret_fort_ici
